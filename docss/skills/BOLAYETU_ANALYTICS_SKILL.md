# BOLAYETU_ANALYTICS_SKILL.md

Analytics are a core business feature.

Every entity should generate metrics.

Organization Metrics:

* Active Competitions
* Active Clubs
* Registered Players

Club Metrics:

* Squad Size
* Match Performance
* Goals

Player Metrics:

* Matches
* Goals
* Assists
* Cards

Design all models with analytics readiness.

Avoid architectures that make future reporting difficult.
